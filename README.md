# Verona IDE Support

Verona adds JSON-friendly support for `*.vomd`, `*.vocs`, and `*.voud` files in both VS Code and JetBrains IDEs.

## Packages

- `verona-vscode/`
  VS Code extension that recognizes Verona files and provides a `Verona: Format As JSON` command.
- `verona-jetbrains/`
  JetBrains plugin that registers a Verona file type and adds `Format As JSON` in the editor context menu.

## Install

Download the packaged artifacts from the [GitHub Releases page](https://github.com/iqb-berlin/verona-ide-support/releases/tag/v1.0.0) for this repository:

- VS Code: `verona-vscode-1.0.0.vsix`
- JetBrains: `verona-jetbrains-1.0.0.jar` or `verona-jetbrains-1.0.0.zip`
